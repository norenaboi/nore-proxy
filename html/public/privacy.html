Privacy Policy

A plain-language summary of what the proxy processes and retains.

Prompt content

Prompts and generated responses pass through the proxy so requests can be served, but their content is not persisted in request logs.

Operational metadata

We retain metadata needed to operate and understand the service. This can include timestamps, model and internal or masked key identity, request status and duration, plus input, output, cache-read, and cache-write token counts.

Errors and security

Diagnostic records may include upstream status, endpoint information, sanitized request headers, and truncated error responses. Authorization values and raw API keys are removed or masked before storage.

Upstream processing

Requests are forwarded to the selected upstream model provider. That provider processes request content under its own terms and privacy practices.
